## Instalación
1. Descargar **[Última versión](https://github.com/shagu/ShaguKill/archive/master.zip)**
2. Descomprima el archivo RAR
3. Copia "ShaguKill" en Directorio-WoW\Interface\AddOns
4. Reiniciar Wow
